## üöÄ Quick Actions

### First Time Setup
```bash
# 1. Run automated setup
setup.bat  # Windows
# OR
chmod +x setup.sh && ./setup.sh  # macOS/Linux

# 2. Get API key
# Visit: https://aistudio.google.com/app/apikey

# 3. Add key to:
# - backend'/.env
# - frontend/.env
```

### Start Development
```bash
# Terminal 1 - Backend
cd backend'
venv\Scripts\activate  # Windows
python main.py

# Terminal 2 - Frontend  
cd frontend
npm start
```

### Test the Feature
1. Open app ‚Üí Learning Hub
2. Select YouTube video
3. Click "Chapters" tab
4. Click "Generate Chapters"
